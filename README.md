# Preço na Mão - Prototype

Protótipo React (Vite) do aplicativo *Preço na Mão* — comparador de preços para cidades pequenas.

## Como rodar localmente

Pré-requisitos:
- Node.js 18+ e npm ou yarn

Passos:
```bash
# clonar ou descompactar o projeto
cd preco-na-mao-prototype

# instalar dependências
npm install

# rodar em modo de desenvolvimento
npm run dev

# construir produção
npm run build
```

O projeto usa Vite + React. É um protótipo front-end sem back-end — funcionalidades como cadastro de preços e compartilhamento são simuladas (alertas).

## Arquivos principais
- `index.html` - entry do Vite
- `src/main.jsx` - bootstrap React
- `src/App.jsx` - protótipo interativo (telas e navegação)
- `src/styles.css` - estilos mínimos
